# Sessions y Autenticación en ExpressJS
Este proyecto es una práctica de la asignatura _Desarrollo de Sistemas Informáticos_ del grado de Ingeniería Informática de la Universidad de La Laguna, en la que crearemos un libro de prueba para así poder poner en práctica las **Sessions y la Autenticación en ExpressJS**, y usar estos métodos para intentar acceder al libro mediante logins, así como poder cambiar la contraseña.
